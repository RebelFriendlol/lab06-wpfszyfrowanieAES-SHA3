# lab06 - WPF Text Encryption (AES & SHA-3)

---

## 🇵🇱 Wersja Polska

### O projekcie
Projekt akademicki zrealizowany w ramach laboratoriów uniwersyteckich. Jest to aplikacja okienkowa (desktopowa) napisana w technologii **WPF (Windows Presentation Foundation)** oparta na platformie **.NET / C#**. 

Głównym celem programu jest bezpieczne szyfrowanie tekstu oraz generowanie sum kontrolnych z wykorzystaniem zaawansowanych mechanizmów kryptograficznych.

### Funkcjonalności:
* Szyfrowanie i deszyfrowanie wiadomości tekstowych przy użyciu symetrycznego algorytmu **AES (Advanced Encryption Standard)**.
* Hashowanie danych i weryfikacja integralności tekstu za pomocą nowoczesnej funkcji skrótu **SHA-3**.
* Graficzny interfejs użytkownika (XAML/WPF) ułatwiający interakcję i zarządzanie kluczami kryptograficznymi.

## UK English Version

### About the Project
An academic application developed as part of university laboratory courses. It is a desktop application built with **WPF (Windows Presentation Foundation)** using **.NET / C#**.

The core purpose of the software is secure text encryption and checksum generation utilizing advanced cryptographic pipelines.

### Features:
* Symmetric text encryption and decryption using the **AES (Advanced Encryption Standard)** algorithm.
* Data hashing and integrity verification powered by the modern **SHA-3** cryptographic hash function.
* Intuitive Graphical User Interface (XAML/WPF) for seamless interaction and cryptographic key handling.
